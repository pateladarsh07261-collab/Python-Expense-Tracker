# 💰 Automated Expense Tracker (Core Python)

## 📌 Project Overview
A lightweight, terminal-based Expense Tracker application built using **Pure Python**. This project demonstrates the practical application of file handling, standard data structures, persistent storage using CSV files, and structured procedural programming in Python.

## 🛠️ Key Features
* **Persistent Data Storage:** Uses Python's native `csv` module to log financial transactions into a structured flat-file database (`expenses.csv`).
* **Expense Logger:** Seamlessly appends transactions with details such as Date, Category, and Transaction Amount.
* **On-Demand Analytics:** Instantly processes historical rows to calculate the live running sum total of all expenditures.
* **Tabular View:** Fetches and streams data from the physical CSV file back into a neatly formatted CLI interface using tab separation delimiters.

## ⚙️ Tech Stack Used
* **Language:** Python 3 (Core modules like `csv`)
* **Database:** CSV File
* **Environment:** VS Code / Terminal / Jupyter Notebook
